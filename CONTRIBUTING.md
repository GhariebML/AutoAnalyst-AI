# Contributing to AutoAnalyst AI

Thank you for contributing. Please keep changes focused, readable, and well documented.

## Rules

- Work on a feature branch, not directly on `main`.
- Write clear commit messages.
- Add or update tests when changing code.
- Keep functions simple and beginner-friendly.
- Open a Pull Request and request review before merging.

## Commit Message Examples

- `feat: add missing values report`
- `fix: handle empty dataframe in profiler`
- `docs: update team workflow`
- `test: add cleaner tests`
