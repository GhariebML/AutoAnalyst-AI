# Team Roles

| # | Role | Responsibilities | Required Skills | Expected Deliverables | Suggested Branch |
|---|---|---|---|---|---|
| 1 | Project Lead / GitHub Manager | Manage repo, issues, PR reviews, milestones, and final integration | Git, GitHub, communication | Clean repo, merged PRs, release checklist | `docs/project-management` |
| 2 | Data Understanding Member | Study dataset context, columns, target variable, and business questions | Pandas basics, domain research | Dataset summary and data dictionary | `feature/data-understanding` |
| 3 | Data Profiling Member | Build missing values, duplicates, dtypes, and profile reports | Pandas, data quality | Profiling functions and report | `feature/data-profiling` |
| 4 | EDA Member | Create summaries, correlations, and visual exploration | Pandas, Matplotlib, Seaborn | EDA notebook and analyzer functions | `feature/eda` |
| 5 | Data Cleaning Member | Handle duplicates, missing data, and inconsistent values | Pandas preprocessing | Cleaning module and tests | `feature/data-cleaning` |
| 6 | Feature Engineering Member | Create useful features and encoding helpers | Pandas, ML basics | Feature builder functions | `feature/feature-engineering` |
| 7 | Classification Modeling Member | Train and compare starter classification models | Scikit-learn | Classification model module | `feature/classification-modeling` |
| 8 | Regression Modeling Member | Train and compare starter regression models | Scikit-learn | Regression model module | `feature/regression-modeling` |
| 9 | Insight & Report Generation Member | Convert outputs into readable insights and Markdown reports | Writing, Markdown, Python | Insight and report modules | `feature/insights-reporting` |
| 10 | Dashboard Developer | Build Streamlit interface and connect pipeline components | Streamlit, UI basics | Interactive dashboard | `feature/dashboard` |
